Git y Github

Alumn@: Sergio Campos Delgado

  1. ¿Que es un sistema de control de versiones?

    Es un mecanismo que procura compatibilizar programas informaticos, Ej: Git

  2. ¿Cual es la diferencia entre git y github?

    Git es un programa diseñado para subir y bajar contenido de servidores, github en cambio es un servicio gratuito para manejar datos enviados por git

  3. ¿Cómo creo una rama en git?

    Usando git branch [nombre]

  4. ¿Cómo clono un repositorio?

    Usando git clone [dirección]

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    Con git pull

  6. ¿Cómo puedo gener un tag?

    Usando git tag

  7. Cómo hago una mezcla de ramas?

    Con git merge

  8. ¿Cómo puedo moverme a un commit especifico?

    Con git checkout

  9. ¿Cómo puedo borrar una rama?

    Con git branch -d

  10. ¿Cómo puedo borrar un tag?

    Con git tag -d

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?

    Usando git status

  12. ¿Para que sirve el comando git remote -v?

    Muestra primero el url y despues el nombre de los repositorios

  13. ¿Cómo le digo a git cual es mi email y correo?

    Usando:
    git config --global user.name "Tu Nombre"
    git config --global user.email "tuemail@example.com"

  14. ¿Cuales son los pasos para conectar git con github?

    Se debe generar una clave ssh, despues usando el codigo de la clave generada se conecta a github en ajustes

  15. ¿Cómo puedo crear un alias en git?

    Usando : git config --global alias.tree "log --graph --decorate --all --oneline"
